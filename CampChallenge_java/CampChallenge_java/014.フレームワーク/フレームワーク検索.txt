Foundation

ZURBが2011年に開発したフレームワークで、ウェブサイトを作ることに適したフレームワークである。特にＥＣサイトの制作に適している。
例えば、Block Gridで商品を並べる機能や、Pricing Tableで価格をやスペックを表示する機能などが充実している。
ヤフーやfacebookなどのサイトもこのフレームを利用している。


Bootstrap
Twitter社が開発した、webサイトやwebページを効率よく開発するためのフレームワーク。一つのページで、ＰＣにも携帯端末にも対応可能である。
ＨＴＭＬとＣＳＳの知識が必要だが、比較的簡単にwebサイトを制作することができる。
機能的には、タグやクラスを指定する事により、ボタンやテーブルを利用できるCSS、ナビゲーションやパンくずリストが用意されたコンポーネント。
スライドショーが利用できるカルーセルのＪＡＶＡＳＣＲＩＰＴ。などのがある。

Web Starter Kit
Googleが提供する、マルチデバイスでのweb開発をサポートする、ボイラープレート（HTMLのひな型）と、開発を補助するツール。
具体的な機能と特徴としては、レスポンシブ ボイラプレート、ウェブ用に最適化された、ウエブサイトのテンプレが用意されている事、JavaScript、CSS、HTML、およびイメージを圧縮して連結し、
ページを小さくする「パフォーマンスの最適化」、複数端末の同期、オフライン サポートなどがあげられる。
